A plugin for enabling Albacross tracking on your WordPress site.

== Description ==

Albacross allows you to monitor which companies visit your website. This plugin simplifies the process of integrating the Albacross tracking code.

== Installation ==

1. Upload `albacross-wordpress-plugin` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > Albacross
4. Enter your client ID that can be obtained from your settings page on https://app.albacross.com/

== Frequently Asked Questions ==

= What is my client ID? =

Your client ID can be found by signing in on https://app.albacross.com/ and going to your settings.
